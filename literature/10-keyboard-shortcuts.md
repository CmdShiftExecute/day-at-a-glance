## Overview

Every major action in the dashboard has a keyboard shortcut. No mouse required for power users.

---

## 🗺️ Navigation

| Key | Action | Description |
|-----|--------|-------------|
| `←` | Previous day | Switch to Yesterday (from Today) or Today (from Tomorrow) |
| `→` | Next day | Switch to Tomorrow (from Today) or Today (from Yesterday) |

---

## ⚡ Actions

| Key | Action | Description |
|-----|--------|-------------|
| `R` | Refresh data | Reload from `my-day-data.xlsx` |
| `I` | Import | Open the Excel file import modal |
| `S` | Settings | Open the settings modal |
| `D` | Toggle theme | Switch between dark and light mode |

---

## ❓ Help

| Key | Action | Description |
|-----|--------|-------------|
| `?` | Help | Open the comprehensive help modal |
| `H` | Help | Alternative key — same action |

> 💡 A small floating keyboard hint appears at the bottom of the dashboard as a reminder. Press `?` to see the full help guide with detailed explanations of every feature.

---

## ⚠️ Notes

- Shortcuts are **disabled when a modal is open** to prevent conflicts with form input
- Shortcuts are **case-insensitive** — both `s` and `S` open settings
- All shortcuts work from any scroll position on the page
